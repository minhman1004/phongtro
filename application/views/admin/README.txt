# Mục đích
Admin là khu vực dành cho người quản trị
- Các cài đặt: 
	- Dữ liệu nền tảng
	- Phân quyền
- Điều khiển
	- Quản lý thông chủ nhà / bất động sản
	- Quản lý thông tin thanh toán / điện nước
	- Bài đăng
	- Người thuê
	- Kiểu bất động sản / Các kiểu thông tin khác
- Bảng chính:
	- Hiển thị các thông số dữ liệu cho người quản trị
